x-img-diff-js demonstration

x-img-diff-js is a JavaScript(Web Assembly) porting project for x-img-diff, which extracts structual information of a bit different 2 images. Click the following "Calculate" button, then rectangle markers are displayed over the images.

Each rectangle stands for:
  • Cyan: Matching region
  • Red: Different parts in the matching region
  • Purple: Keypoints region which is not include in any matching regions
